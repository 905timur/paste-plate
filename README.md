# Paste Plate

A cross-platform clipboard manager built with Rust and egui.

## Features

- Cross-platform support (Linux, macOS, Windows)
- Wayland and X11 support on Linux
- Image clipboard support

## Installation

```bash
cargo install paste-plate
```

Or build from source:

```bash
git clone https://github.com/yourusername/paste-plate.git
cd paste-plate
cargo build --release
```

## Usage

```bash
paste-plate
```
